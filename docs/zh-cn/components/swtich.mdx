## swtich / 开关

| 属性         | 描述     | 类型                  | 可选值               | 默认    |
| ------------ | -------- | --------------------- | -------------------- | ------- |
| **checked**  | 是否选中 | `boolean`             | `'true','false'`     | `false` |
| **size**     | 开关大小 | `string`              | `'true','false'`     | `false` |
| **disabled** | 禁用交互 | `boolean`             | `'true','false'`     | `false` |
| **change**   | 开关事件 | `SwitchEvent`         | `-`                  | `-`     |
| ...          | 原生属性 | `LabelHTMLAttributes` | `'class','name',...` | `-`     |

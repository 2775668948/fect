## spacer / 间距

| 属性  | 描述       | 类型                | 可选值             | 默认 |
| ----- | ---------- | ------------------- | ------------------ | ---- |
| **x** | 水平轴间距 | `'string','number'` | `-`                | `1`  |
| **y** | 垂直轴间距 | `'string','number'` | `-`                | `1`  |
| ...   | 原生属性   | `HTMLAttributes`    | `'class','id',...` | `-`  |

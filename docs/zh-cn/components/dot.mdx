## Dot / 点

| 属性     | 描述     | 类型             | 可选值                                     | 默认值    |
| -------- | -------- | ---------------- | ------------------------------------------ | --------- |
| **type** | 点的属性 | `'string'`       | `'default', 'success', 'warning', 'error'` | `default` |
| ...      | 原生属性 | `HTMLAttributes` | `'class','id',...`                         | `-`       |

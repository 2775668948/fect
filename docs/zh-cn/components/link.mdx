## Link / 链接

| 属性          | 描述           | 类型                   | 可选值                 | 默认    |
| ------------- | -------------- | ---------------------- | ---------------------- | ------- |
| **href**      | 链接地址       | `string`               | `-`                    | `-`     |
| **color**     | 是否高亮显示   | `boolean`              | `'true','false'`       | `false` |
| **underline** | 是否显示下划线 | `boolean`              | `'true','false'`       | `false` |
| **block**     | 是否为块级元素 | `boolean`              | `'true','false'`       | `false` |
| ...           | 原生属性       | `AnchorHTMLattributes` | `'rel', 'target', ...` | `-`     |

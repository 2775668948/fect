<br/>

### Quick start

1. run `yarn add @fec-ui/vue` or `npm install @fect-ui/vue` install it

2. import into project

```js
import { createApp } from 'vue'
import App from './App.vue'
import FectUI from '@fect-ui/vue'
import '@fect-ui/vue/dist/fect-ui.css'

createApp(App)
  .use(FectUI)
  .mount('#app')
```
